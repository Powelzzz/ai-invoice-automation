# AGENTS.md

## Project goal
Build an AI invoice processing automation for a technical assessment.

## Deliverables
- README.md
- technical and non-technical user docs
- n8n workflow notes
- prompt file
- sample payload
- demo script
- interview Q&A
- exported workflow JSON to be added later

## Stack
- n8n for workflow automation
- Google Drive for invoice intake
- OCR.Space for OCR
- OpenRouter for invoice field extraction
- Google Sheets for record storage
- Gmail for notifications

## Rules
- Keep the solution simple and demo-friendly
- Prefer practical over overengineered
- Do not invent unsupported integrations
- Assume credentials are configured manually in n8n
- Documentation should be concise and professional
- Focus on explainability for a live interview

## Done means
- Repo is complete and presentable
- Workflow steps are documented node-by-node
- Prompt is production-ready
- Demo script is usable as-is
